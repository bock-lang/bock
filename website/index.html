Bock

One language. Many targets.

Bock is a feature-declarative, target-agnostic programming language. Write your application's logic, types, and effects once — Bock emits idiomatic source code in JavaScript, TypeScript, Python, Rust, or Go.

View on GitHub Read the docs

Static types, inferred

Bidirectional type inference with explicit annotations only when you want them.

Effects on every function

I/O, network, randomness, and time are tracked by the compiler — never hidden.

Targeted output, not a runtime

Generated code reads like hand-written code. There's no Bock VM.
