# expense-tracker

Rust output generated by [Bock](https://bock-lang.org) (project mode).

## Run

```sh
cargo run
```

## Test

Tests are generated as `cargo test` tests:

```sh
cargo test
```

Formatting is `rustfmt` (`cargo fmt`); the output is rustfmt-clean.

> Generated by `bock build`. Re-running the build regenerates this output.
