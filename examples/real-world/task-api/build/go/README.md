# task-api

Go output generated by [Bock](https://bock-lang.org) (project mode).

## Run

```sh
go run .
```

## Test

Tests are generated as `go test` tests (stdlib `testing`):

```sh
go test ./...
```

Formatting is `gofmt` (`go fmt ./...`); the output is gofmt-clean.

> Generated by `bock build`. Re-running the build regenerates this output.
